Entornos de desarrollo para Android

Android Studio

Plataforma: Multiplataforma.
Web: developer.android.com
Lenguaje de programación: Java, Kotlin.
Coste: gratuito.

Es el entorno de desarrollo oficial para Android. Existen multitud de librerías y código disponible en general para desarrollar en Java y con Android Studio.

AIDE (Android IDE)


Plataforma: Android.
Web: android-ide.com
Lenguaje de programación: Java.
Coste: gratuito para uso personal.

Entorno de desarrollo bajo Android, lo cual hace pensar en la necesidad de un teclado físico y una pantalla grande, como una tablet o un chromebook. Se puede desarrollar con las mismas librerías que con Android Studio. Se integra con GitHub.

DroidScript


Plataforma: Android (también se puede editar el código en el navegador).
Web: droidscript.org
Lenguaje de programación: JavaScript.
Coste: gratuito con algunos módulos avanzados de pago.

Entorno de desarrollo bajo Android, aunque se puede comenzar la app en el dispositivo y después pasar a desarrollar bajo cualquier navegador. Tiene un plugin que es capaz de crear un apk a partir del código desarrollado.

B4x


Plataforma: Android, iOS, Windows, Arduino.
Web: b4x.com
Lenguaje de programación: Variante moderno de Basic, tipo Visual Basic.
Coste: gratuito.

Entorno de desarrollo multiplataforma que permite crear aplicaciones multiplataforma (Windows, Linux, Mac, Android...) de manera que excepto el código dedicado a la interfaz de usuario, puede ser reutilizado de un proyecto a otro.

MIT AppInventor


Plataforma: Navegador.
Web: appinventor.mit.edu
Lenguaje de programación: JavaScript.
Coste: gratuito.

Entorno de desarrollo HTML5 que permite crear aplicaciones para Android juntando bloques (graficamente). Es capaz de crear un apk que es posible descargar al ordenador local. Puede conectarse a un emulador o a un dispositivo real.
